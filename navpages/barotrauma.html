Barotrauma Text Splitter

A modding tool for Barotrauma, a 2D co-op submarine simulator. This tool can be used to extract attributes from XML elements. It was built mainly on Java using JAXP and Swing. For more information, visit my GitHub repository. There is also a web version which aims to offer more convenient access to the tool without installing Java.

And it feels great when you can finally put something you learned into application.
